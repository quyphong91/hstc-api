# Privacy Policy - HSTC API & Custom GPT

**Last Updated:** February 14, 2026

## Overview

This Privacy Policy describes how the HSTC (Vietnamese HS Code Classification) API and Custom GPT handle user data.

## Information We Collect

### Search Queries
- **What:** Product descriptions and search keywords you submit
- **Purpose:** To search through HS Code Explanatory Notes and return classification results
- **Storage:** Processed in real-time, not stored permanently

### Technical Data
- **IP Addresses:** Collected temporarily for rate limiting (anti-abuse)
- **Retention:** Discarded after 1 hour
- **Purpose:** Prevent API abuse, not for user tracking

### What We DON'T Collect
- No personal information (name, email, phone)
- No user accounts or profiles
- No cookies or tracking pixels
- No location data
- No browsing history

## How We Use Information

1. **Processing Searches:** Your query is sent to our API to search HS Code notes
2. **Returning Results:** We return matching HS codes with explanatory text
3. **Rate Limiting:** IP addresses used to prevent abuse (1000 requests/hour)
4. **No Marketing:** We do not use data for marketing or advertising

## Data Storage & Security

- **Queries:** Not stored beyond request processing (~50ms)
- **Rate Limit Data:** Stored in memory for 1 hour, then discarded
- **Security:** HTTPS encryption, Helmet.js security headers
- **Hosting:** Vercel (subject to Vercel's privacy policy)

## Third-Party Services

### Vercel (Hosting)
- Our API is hosted on Vercel
- Subject to [Vercel's Privacy Policy](https://vercel.com/legal/privacy-policy)

### OpenAI / ChatGPT
- When using our Custom GPT, queries are processed by ChatGPT
- Subject to [OpenAI's Privacy Policy](https://openai.com/policies/privacy-policy)

## Data Sharing

- **We do not sell, rent, or share your data with third parties**
- **No data brokers**
- **No advertising networks**
- **Open source:** API code is public on GitHub, transparent

## User Rights

Since we do not collect or store personal data:
- No data to access, correct, or delete
- No user accounts to manage
- Queries are ephemeral (processed and forgotten)

## Children's Privacy

- We do not knowingly collect data from children

## International Users

- No specific data residency guarantees
